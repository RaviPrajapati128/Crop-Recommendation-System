# 🌾 Crop Recommendation System

## 📖 Overview
This project uses **Machine Learning (K-Nearest Neighbors - KNN)** to recommend the most suitable crop based on **soil nutrients (N, P, K, pH)** and **climate conditions (temperature, humidity, rainfall)**.  
It is built with **Python** and **Streamlit**, providing an interactive dashboard for farmers, students, and researchers.

**Model Accuracy:** ✅ 97.04% (KNN Classifier)

---

## 🚀 Features
- 📊 Data-driven crop recommendations  
- 🌱 Simple and interactive Streamlit interface  
- 🌍 Supports sustainable farming practices  
- 🖥️ Easy to deploy and extend  

---

## 🛠️ Tech Stack
- **Python**  
- **Streamlit** (for UI)  
- **scikit-learn / pandas / NumPy** (for ML & data handling)
